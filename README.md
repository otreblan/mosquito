# MQTT test

## Dependencies

* `bash-completion`
* `cmake`
* `libmosquittopp-dev`

## Build
``` bash
git clone https://github.com/otreblan/mosquito
cd mosquito

mkdir -p build
cd build

cmake ..
make
```
